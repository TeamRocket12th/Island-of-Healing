## 功能介紹
### < 使用者 >
- 註冊登入
- 個人資訊編輯更新
- 閱讀文章
- 收藏文章
- 追蹤作家
- 付費訂閱平臺，解鎖付費文章
- 透過「藍新金流」刷卡付款
- 與平台設定的AI角色聊天
- 收到站內信通知


### < 作家 >
- 更改個人頭銜 & 簡介
- 發表文章
- 設定文章為免費/付費
- 查看追蹤者人數
- 查看文章流量數據
- 查看平台分潤金額
