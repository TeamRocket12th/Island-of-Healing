## 平台特色



## 功能介紹
### < 使用者 >
- 註冊登入
- 個人資訊編輯更新
- 閱讀文章
- 收藏文章
- 給予文章愛心
- 在文章下方留言
- 追蹤作家
- 付費訂閱平臺，解鎖付費文章
- 透過「藍新金流」刷卡付款
- 查看歷史訂單
- 與平台設定的AI角色聊天
- 收到站內信通知
- 向平台申請成為作家


### < 作家 >
- 更改個人頭銜 & 簡介
- 發表文章
- 設定文章為免費/付費
- 查看追蹤者人數
- 查看文章流量數據
- 查看平台分潤金額

### < 平台管理員 >
- 審核作家申請是否通過
- 審核文章是否通過並發表
- 取消作家身份
- 新增 / 取消 精選文章
- 新增 / 編輯作家分潤費用單
- 發送站內信通知
  - 通知使用者追蹤的作家已發表新文章
  - 通知作家申請成功
  - 通知文章審核成功
